# **App Name**: PantryPilot

## Core Features:

- User Authentication: Secure user login and registration with email/password and Google accounts using Firebase Authentication for managing personal inventory and meal plans.
- Grocery Inventory Management: Users can add, edit, and delete grocery items with details like name, quantity, unit, optional expiry date, and category. Display the inventory with visual expiry indicators (e.g., color-coded based on remaining shelf life) and allow sorting.
- In-App Inventory Alerts: Display prominent alerts within the dashboard for low-stock items or items expiring soon based on user-defined thresholds, providing a clear overview of items needing attention.
- Meal Planning & Recipe Association: Users can plan meals daily, associating custom or saved recipes with specific meal slots (breakfast, lunch, dinner, snacks) and defining required ingredients for each plan.
- AI Meal Suggester Tool: This AI tool suggests meal ideas from the user's saved recipes, intelligently comparing ingredient requirements against the current inventory and highlighting missing or low-stock items for each suggestion.
- Smart Grocery List Generation: Automatically generates a shopping list by identifying missing ingredients from planned meals and low-stock inventory items. Users can manually add/remove items and trigger bulk additions directly to the list.

## Style Guidelines:

- Primary color: A deep, sophisticated green (#48784A) for elements related to growth, freshness, and core actions, symbolizing the app's connection to natural food and organized living. This color is moderately saturated to evoke calmness.
- Background color: A very light, desaturated green-white (#F3F6F3). This serves as a clean canvas, providing a sense of spaciousness and adhering to the user's preference for a white base, while subtly reinforcing the fresh green palette.
- Accent color: A vibrant chartreuse (#A7DE66). This brighter, more saturated green will be used for interactive elements, highlights, and status indicators, offering an energetic contrast that draws attention and guides the user.
- Headline and body font: 'Inter' (sans-serif) for its modern, clean, and highly readable characteristics, suitable for various UI elements and content types. It aligns well with a responsive Material Design approach.
- Utilize a comprehensive set of Material Design Icons for clarity and consistency across all functional elements, ensuring immediate user recognition and adherence to a modern aesthetic.
- Implement a responsive grid-based layout for the main dashboard and key content areas, ensuring optimal display and usability across different screen sizes. Emphasize a clean, card-based approach for organizing information such as daily meals, inventory alerts, and quick-add features.
- Incorporate subtle transitions for navigation, loading states, and content updates. Implement mild hover effects for interactive components, enhancing user engagement without introducing distractions.
